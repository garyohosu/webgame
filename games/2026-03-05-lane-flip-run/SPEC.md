# レーンフリップ・ラン
- Date: 2026-03-05
- Slug: lane-flip-run

## 企画（Producer）
- 1メカニクス：タップで左右レーンを即時切替し、障害物を避けつつコインを拾う。
- フック（1行）：迷わず1タップ、右か左か――反射だけで60秒を走り切る。
- 勝ち条件：60秒の制限時間を生き残る。
- 負け条件：障害物に3回ぶつかってライフが0になる。
- 1プレイ時間：60秒以内

## ルール（Designer）
- プレイヤーは常に左か右のどちらかのレーンにいる。
- 画面上から障害物とコインが落ちてくる。
- タップするたびにプレイヤーのレーンが左右反転する。
- コイン獲得で +10 点、障害物接触でライフ -1（短い無敵あり）。
- 60秒経過またはライフ0でゲーム終了。

## UI（Designer）
- ボタン：スタートボタン 1つ
- 表示：スコア / 残り時間 / ハイスコア / ライフ
- 操作：タップ（pointerdown）でレーン切替

## 難易度（Designer）
- 0-30秒：落下速度ゆるめ、障害物比率低め。
- 30-45秒：落下速度上昇、障害物の出現密度アップ。
- 45-60秒：さらに速度上昇、障害物比率高めで判断時間が短くなる。
