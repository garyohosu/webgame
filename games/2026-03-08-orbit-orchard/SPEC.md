# オービットオーチャード
- Date: 2026-03-08
- Slug: orbit-orchard

## 企画（Producer）
- 1メカニクス：中央のコアをタップで回転させ、落ちてくるフルーツを対応色のゾーンで受け止める。
- フック（1行）：タップ1回で運命が変わる、色合わせ反射ゲーム。
- 勝ち条件：60秒でできるだけ多くキャッチしてスコアを伸ばす。
- 負け条件：ミスでライフが0になる。
- 1プレイ時間：60秒以内

## ルール（Designer）
- タップするたびに受け皿リングが90度回転する。
- 落下フルーツ（4色）は、同色ゾーンで受けると+1点。
- 色が合わないゾーンで受けるか取り逃すとライフ-1。
- ライフは3で開始し、0で即ゲームオーバー。
- 60秒経過またはライフ0で終了し、ハイスコアを保存する。

## UI（Designer）
- ボタン：スタートボタン1つ
- 表示：スコア / 残り時間 / ハイスコア / ライフ
- 操作：タップ（pointerdown）でリング回転

## 難易度（Designer）
- 0-30秒：落下速度ゆるめ、同時出現1個
- 30-45秒：落下速度1.2倍、出現間隔短縮
- 45-60秒：落下速度1.4倍、2個同時出現の確率追加
