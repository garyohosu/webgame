# パルスストッパー
- Date: 2026-03-04
- Slug: pulse-stopper

## 企画（Producer）
- 1メカニクス：拡大するリングを「ジャスト範囲」でタップして止めるタイミング判定
- フック（1行）：気持ちいい“ジャスト”を60秒で何回決められるかを競う集中ゲーム。
- 勝ち条件：制限時間内にスコアをできるだけ伸ばす（自己ベスト更新）
- 負け条件：時間切れで終了（ミスは減点）
- 1プレイ時間：60秒以内

## ルール（Designer）
- スタート後、中心からリングが連続で広がる。
- 画面中央のターゲット円にリングが重なった瞬間にタップすると成功。
- 成功で加点、誤タップやズレたタイミングは減点（最低0点）。
- 連続成功でコンボ倍率が上がり、得点効率が上がる。
- 60秒経過で終了し、ハイスコアを保存する。

## UI（Designer）
- ボタン：スタートボタン1つ
- 表示：スコア / 残り時間 / ハイスコア / コンボ / 判定メッセージ
- 操作：タップ（pointerdown）

## 難易度（Designer）
- 0-30秒：リング速度は標準、判定幅はやや広め。
- 30-45秒：リング速度アップ、判定幅を少し狭く。
- 45-60秒：リング速度さらにアップ、判定幅を最小に。
