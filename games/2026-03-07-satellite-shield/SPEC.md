# サテライトシールド
- Date: 2026-03-07
- Slug: satellite-shield

## 企画（Producer）
- 1メカニクス：3レーンの落下物に対して、タップしたレーンだけ一瞬シールドを張る
- フック（1行）：守るレーンの選択を1秒ごとに迫られる、60秒の宇宙防衛パズルアクション。
- 勝ち条件：60秒経過時点でコア耐久（HP）が1以上残っている
- 負け条件：60秒以内にHPが0になる
- 1プレイ時間：60秒以内

## ルール（Designer）
- 上から3レーンにデブリが落下し、下端のコアに到達するとHPが1減る。
- 画面下のレーンボタンをタップすると、そのレーンに短時間シールドが展開される。
- シールド展開中にデブリを受け止めるとスコア+1、同時にデブリを消去する。
- シールドは短いクールダウンがあり、連打だけでは防ぎ切れない。
- 60秒生存でクリア。HPが尽きたら即ゲームオーバー。

## UI（Designer）
- ボタン：スタート1つ、レーンシールド3つ
- 表示：スコア / 残り時間 / ハイスコア / HP
- 操作：タップ（pointerdown）

## 難易度（Designer）
- 0-30秒：落下速度はゆるめ、同時出現数は最大2。
- 30-45秒：落下速度を約1.2倍、出現頻度を上げる。
- 45-60秒：落下速度を約1.4倍、3レーン同時圧力を増やす。
